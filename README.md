# ScaffoldingTentShop

Banner ASCII + descrição

## Funcionalidades
- Lojas físicas (scaffolding + carpet)
- Hologramas configuráveis
- 6 GUIs (navegar, comprar, vender, configurar, adicionar produto, admin)
- Economia via Vault
- Sistema de expiração de lojas

## Requisitos
- Minecraft 1.21+ (Paper/Spigot)
- Vault
- Plugin de economia (EssentialsX, CMI, etc.)

## Instalação
- Baixar o .jar, colocar na pasta plugins, reiniciar

## Como Usar
- Criar loja (colocar carpet em cima de scaffolding)
- Configurar (/lojinha)
- Comprar/Vender (click esquerdo/direito)

## Comandos
- /lojinha - Abrir GUI das suas lojas
- /lojinha admin - Painel administrativo

## Permissões
- playsershops.admin
